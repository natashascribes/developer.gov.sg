Components that facilitate communication between systems and applications, e.g. through API and traffic management.

APEX

APEX is a self-service API platform that acts as a centralised and secure API gateway for the Government.

Cloud File Transfer

Cloud File Transfer is a lightweight, fully managed, API-driven service that allows agencies or businesses to transfer files easily, securely and efficiently at scale.
